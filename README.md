## 📌 Sobre o Projeto

Portfólio moderno e responsivo desenvolvido para apresentar os trabalhos de **Isadora**, designer gráfica especializada em branding, social media e identidade visual.

O site foi criado para transmitir profissionalismo, destacar projetos reais e facilitar o contato com clientes.

---

## 🚀 Tecnologias

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- React Router DOM
- Lucide React

---

## ✨ Funcionalidades

- Layout responsivo (mobile, tablet e desktop)
- Animações suaves ao rolar a página
- Navbar fixa com efeito blur
- Seção de apresentação profissional
- Portfólio com projetos reais
- Lista de ferramentas dominadas
- Botões para redes sociais
- Formulário com redirecionamento para WhatsApp
- Página 404 personalizada
